CSS (Cascading Style Sheets) é uma linguagem essencial para o desenvolvimento web, responsável pela formatação e layout das páginas. Enquanto o HTML estrutura o conteúdo, o CSS define a aparência visual, controlando elementos como cores, fontes, espaçamento e disposição. Com recursos como media queries, permite criar designs responsivos, adaptando-se a diferentes tamanhos de tela.

HTML (HyperText Markup Language) é a linguagem de marcação fundamental para a construção de páginas web. Ele define a estrutura e o conteúdo de uma página, utilizando uma série de elementos, como títulos, parágrafos, listas, links, imagens e formulários. Com a organização dessas tags, o HTML cria a base sobre a qual o CSS e o JavaScript podem ser aplicados, permitindo a formatação e interatividade. Além disso, o HTML é responsável por tornar os sites acessíveis aos motores de busca e a leitores de tela, garantindo uma boa experiência para todos os usuários.

Arduino é uma plataforma de prototipagem eletrônica de código aberto que permite a criação de projetos interativos, combinando hardware e software de maneira simples e acessível. Utilizando uma placa de circuito e uma linguagem de programação baseada em C/C++, o Arduino facilita a construção de dispositivos como sensores, motores e luzes, que podem ser controlados por meio de comandos codificados.
